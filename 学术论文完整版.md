# 基于 MEKF、LADRC 与在线惯量辨识的卫星姿态控制综合仿真平台研究

## 摘要

针对卫星姿态控制系统在模型不确定性、外部扰动、传感器噪声及惯量参数偏差共同作用下的控制与估计问题，本文设计并实现了一套面向刚体卫星的姿态控制综合仿真平台，并在该平台上开展姿态估计、控制器比较、参数优化及在线惯量辨识研究。本文采用四元数建立卫星姿态运动学模型，基于欧拉方程建立三轴刚体姿态动力学模型，并在仿真中引入外部扰动力矩和执行机构饱和约束。针对测量链路，本文构建了包含视场筛选、向量测量噪声和 Wahba 姿态解算的星敏感器模型，同时建立了含偏置随机游走的陀螺仪模型。针对姿态估计问题，本文实现了基于乘法扩展卡尔曼滤波器的姿态与陀螺偏置估计方法，并进一步实现了用于惯量联合估计的增广状态 MEKF。针对控制问题，本文实现了基于四元数误差的比例微分控制器与线性自抗扰控制器，并设计了基于多优化器横向比较的 PD 参数自动调优框架。针对参数不确定问题，本文提出并实现了基于 ESO + RLS 和基于增广状态 MEKF 的两类对角惯量在线辨识方法。

在实验方面，本文利用统一平台对 PD 与 LADRC 的控制性能进行对比，并对五类优化器在 PD 参数整定问题上的表现进行分析，同时对两类在线惯量辨识方法的阶段性效果进行比较。实验结果表明：在当前代表性场景下，PD 控制器具有更快的动态响应速度，而 LADRC 在稳态误差、控制能耗和饱和抑制方面表现更优；在参数优化问题中，PSO 和 Nelder-Mead 均能获得较优的 PD 参数；在惯量辨识方面，两类方法均能实现对角惯量的在线修正，其中增广状态 MEKF 在当前实验条件下表现出更小的最终误差范数。为提高实验效率，本文还设计了面向调参任务的轻量化指标评估路径，显著降低了优化器横向比较的计算开销。研究结果表明，本文所构建的平台能够较完整地复现卫星姿态控制中的“动力学—测量—估计—控制—辨识”闭环过程，并具备良好的研究扩展能力。

## 关键词

卫星姿态控制；四元数；MEKF；LADRC；PD 控制；在线惯量辨识；递推最小二乘；仿真平台

## 1. 引言

卫星姿态控制系统是航天器完成稳定指向、对地观测、空间通信、深空探测以及在轨服务任务的重要支撑，其控制精度和动态性能直接影响任务成败。随着空间任务复杂度不断提高，卫星姿态控制不再仅仅是理想模型下的稳定化问题，而是需要在不确定动力学参数、外部扰动力矩、传感器噪声、执行器约束及多源信息耦合条件下实现快速、稳定和高精度控制的问题。尤其是在高精度遥感、窄波束通信和长时间稳定观测等任务中，微小姿态偏差都可能带来明显的任务性能退化，因此开展面向复杂不确定条件的姿态控制研究具有重要意义。

从研究方法上看，姿态控制问题通常包含动力学建模、姿态测量、状态估计、控制律设计和参数辨识等多个相互关联的环节。若仅在其中某一个局部模块上开展研究，往往难以反映整个闭环系统的真实表现。例如，控制器的性能不仅由控制律决定，还受到姿态估计误差、扰动补偿能力及参数模型准确性的影响；同样，参数辨识效果也依赖于控制激励条件和滤波结果。因此，建立一套统一、可复现、可扩展的姿态控制综合仿真平台，对于控制算法比较、估计方法验证和参数适应研究具有重要的理论价值与工程价值。

在控制方法方面，PD 控制由于结构简单、实现方便、参数物理意义明确，长期以来一直是航天器姿态控制中的经典基线方法。然而，PD 控制本质上依赖较准确的模型参数，对持续扰动和模型不确定性的鲁棒性有限。相比之下，自抗扰控制通过将系统中未知的内外扰动统一视为总扰动，并利用观测器进行实时估计和补偿，在非精确建模条件下表现出更强的鲁棒控制潜力。在姿态估计方面，四元数已成为航天器姿态表示的主流形式，而基于误差状态建模的 MEKF 则是姿态估计中应用广泛且数值稳定的经典方法。

此外，卫星惯量参数作为动力学模型中的关键参数，其偏差会直接影响控制器设计效果与滤波器建模精度。由于卫星构型变化、载荷展开、燃料消耗及建模误差等因素，惯量参数可能与设计值存在差异，因此在线惯量辨识与参数自适应问题同样值得研究。基于上述背景，本文围绕卫星姿态控制综合仿真平台构建这一核心任务，完成动力学建模、测量仿真、状态估计、控制器实现、参数优化与在线辨识等模块的统一设计，并在该平台上开展阶段性实验分析。

从研究问题角度看，本文希望回答如下几个核心问题：第一，在考虑测量噪声、扰动和执行器约束的条件下，能否构建一套完整的姿态控制闭环仿真平台；第二，在同一评价框架下，PD 与 LADRC 分别表现出怎样的控制性能侧重；第三，在线惯量辨识是否能够在当前平台上稳定工作，并为控制器内部模型更新提供依据；第四，在多优化器调参问题中，如何兼顾实验效率与评价一致性。围绕这些问题，本文的工作不仅关注算法本身的可行性，也关注实验流程、结果复现和论文表达的统一性。

从实现难点上看，本研究面临如下挑战：一是姿态系统具有非线性和几何约束特性，四元数的传播与误差表达必须保持一致；二是星敏感器与陀螺仪测量特性差异较大，测量仿真与滤波器设计之间存在明显耦合；三是惯量参数属于慢变量且不被姿态量测直接观测，在线辨识对系统激励和噪声抑制能力要求较高；四是多优化器横向比较需要大量重复仿真调用，若软件结构设计不合理，实验效率将成为后续研究的主要瓶颈。

本文的主要工作可概括如下：

1. 建立了基于四元数的卫星姿态运动学模型和三轴刚体姿态动力学模型，并引入扰动和力矩饱和约束；
2. 构建了包含 Wahba 解算的星敏感器模型和含偏置随机游走的陀螺仪模型；
3. 实现了用于姿态与偏置估计的 MEKF 以及用于惯量联合估计的增广状态 MEKF；
4. 实现了 PD 控制器和 LADRC 控制器，并建立统一的控制对比流程；
5. 设计了基于多优化器横向比较的 PD 自动调参框架；
6. 实现了基于 ESO + RLS 与基于增广状态 MEKF 的两类对角惯量在线辨识方法；
7. 构建了图像、表格和 Markdown 报告自动输出机制，并通过轻量化评估路径提升优化实验效率。

从阶段性创新点来看，本文当前工作的特色主要体现在以下几个方面：其一，将动力学、传感器、姿态估计、控制器、在线辨识与结果输出统一集成在同一实验框架中，而非停留在单一算法验证层面；其二，在控制比较之外，同时引入参数自动优化和参数在线辨识两个研究维度，使平台更贴近实际研究需求；其三，设计了基于 `result_mode='metrics'` 的轻量化调参路径，在保证评价口径一致的前提下显著提高了优化效率；其四，支持将惯量辨识结果反馈到 ADRC 内部模型，使控制器与辨识器之间形成可研究的联动关系。

## 2. 相关工作

在姿态表示与估计方面，四元数由于不存在欧拉角奇异性且适合处理大姿态机动，已广泛用于航天器姿态控制与姿态估计。针对四元数姿态估计，扩展卡尔曼滤波、无迹卡尔曼滤波及其改进方法均有大量研究。其中，MEKF 通过采用“名义姿态 + 误差状态”的结构，在保持单位四元数约束和提高线性化处理合理性之间取得了较好平衡，因此成为航天器姿态估计中的经典方法。

在控制方法方面，四元数反馈控制、最优控制、滑模控制、自适应控制与模型预测控制均被用于航天器姿态控制研究。PD 控制器由于结构简单、调参直观，仍是许多实际系统中的基准控制方法。自抗扰控制则通过扩张状态观测器对未知扰动进行估计和补偿，使控制器对模型不确定性的依赖程度降低，在扰动环境下具有较强吸引力。近年来，LADRC 在伺服控制、飞行器控制和电机控制等领域已有较多应用，但在航天器姿态控制综合仿真平台中的系统化集成研究相对较少。

在参数辨识方面，递推最小二乘方法适用于可线性化参数回归的在线辨识问题，因其计算量低、实现清晰而具有工程实用价值；增广状态滤波方法则可以在统一概率框架下完成状态与参数联合估计。对于航天器惯量参数而言，由于其不被姿态传感器直接观测，在线辨识问题通常需要足够激励条件和合理的估计框架支撑。

现有文献虽然分别对姿态估计、鲁棒控制和惯量辨识开展了研究，但将这些内容统一纳入一个可反复迭代、可自动输出实验结果的综合仿真平台中的工作相对有限。本文的研究重点之一，正是通过统一平台将上述问题贯通起来，为系统化实验研究提供基础。

## 3. 系统模型与问题描述

### 3.1 四元数姿态运动学模型

设卫星姿态四元数为

`q = [q0, q1, q2, q3]^T`

其中 `q0` 为标量部，`[q1, q2, q3]^T` 为向量部。设角速度向量为

`w = [wx, wy, wz]^T`

则姿态运动学方程可写为

`qdot = 0.5 * Omega(w) * q`

其中

`Omega(w) = [[0, -wx, -wy, -wz], [wx, 0, wz, -wy], [wy, -wz, 0, wx], [wz, wy, -wx, 0]]`

为了保证数值积分精度，本文采用四阶 Runge-Kutta 方法进行离散传播，并在每次更新后对四元数进行归一化处理，以满足单位模约束。

### 3.2 刚体姿态动力学模型

本文将卫星视为刚体，转动动力学满足欧拉方程：

`J * wdot = u - w x (Jw) + d`

其中 `J` 为转动惯量矩阵，`u` 为控制力矩，`d` 为外部扰动力矩。若 `J` 已知，则有

`wdot = J^(-1) * (u - w x (Jw) + d)`

当前研究主要考虑对角惯量情形，即

`J = diag(Jxx, Jyy, Jzz)`

这一处理既符合当前控制器结构，也有利于在线惯量辨识问题的简化与稳定实现。

### 3.3 扰动与执行器约束

为增强仿真模型的现实意义，本文在动力学中叠加常值扰动和随机扰动。常值扰动用于近似表征长期未建模偏置力矩，随机扰动用于模拟高频环境不确定项。控制器输出进一步经过力矩限幅：

`u_sat = clip(u, -u_max, u_max)`

该约束不仅影响闭环响应，也会影响观测器与辨识器的可用激励水平。

### 3.4 星敏感器与陀螺仪测量模型

#### 3.4.1 星敏感器模型

星敏感器模型由星表生成、视场筛选、方向测量噪声注入和 Wahba 姿态解算组成。设惯性系参考星向量为 `r_i`，机体系测量向量为 `v_i`，则姿态解算问题可归结为 Wahba 问题：

`min_R sum(a_i * ||v_i - R * r_i||^2)`

本文采用基于奇异值分解的求解方法获得姿态矩阵，并转换为姿态测量四元数。

#### 3.4.2 陀螺仪模型

陀螺仪测量模型表示为

`w_gyro = w + b + n`

其中 `b` 为零偏项，`n` 为高斯白噪声。零偏进一步采用随机游走模型：

`b(k+1) = b(k) + eta_b`

其中 `eta_b` 为小幅偏置扰动。

## 4. 姿态估计与在线参数辨识方法

### 4.1 MEKF 姿态与偏置估计

对于四元数姿态系统，若直接采用常规加法型 EKF，会带来单位模约束处理复杂和线性化误差放大的问题。为此，本文采用乘法扩展卡尔曼滤波方法，将姿态估计分解为名义四元数和误差状态。本文首先实现了 `MEKFBiasOnly`，其误差状态为

`x = [δθ, δb]^T`

其中 `δθ` 为小角度姿态误差，`δb` 为陀螺偏置误差。预测阶段，名义姿态按照去偏置后的角速度进行传播：

`q_hat(k+1|k) = q_hat(k|k) ⊗ dq(omega_meas - b_hat, dt)`

协方差传播则采用离散线性误差模型：

`P(k+1|k) = Phi * P(k|k) * Phi^T + Q`

更新阶段，通过姿态测量四元数与名义四元数构造姿态残差，计算卡尔曼增益：

`K = P * H^T * (H * P * H^T + R)^(-1)`

并用误差状态更新偏置估计和名义姿态。该结构能够兼顾姿态表示的几何特性与滤波更新的数值稳定性。

### 4.2 增广状态 MEKF

为了将惯量参数纳入统一估计框架，本文在 `MEKFBiasOnly` 的基础上实现了 `MEKF_Augmented`。其增广误差状态为

`X = [δθ, δb, ΔJ]^T`

其中 `ΔJ` 表示对角惯量偏差。本文采用随机游走模型描述惯量缓慢变化，并在预测阶段通过动力学模型构造惯量误差对姿态误差传播的影响。由于惯量参数不被姿态测量直接观测，因此其修正依赖于状态与参数之间的交叉协方差。该方法在理论上具有更统一的估计框架，但对激励条件与协方差设计更敏感。

### 4.3 基于 ESO + RLS 的对角惯量在线辨识

在对角惯量假设下，欧拉方程可以写成线性回归形式：

`y = Phi * theta`

其中

- `theta = [Jxx, Jyy, Jzz]^T`
- `y = u + d_hat`
- `Phi = Phi(omega, wdot)`

式中 `d_hat` 为由 LADRC 中扩张状态观测器输出换算得到的等效扰动力矩估计。RLS 更新公式为：

`K_rls = P * Phi^T * (lambda * I + Phi * P * Phi^T)^(-1)`

`theta(k+1) = theta(k) + K_rls * (y - Phi * theta(k))`

`P(k+1) = (I - K_rls * Phi) * P / lambda`

考虑到角加速度由离散差分获得，容易受到高频噪声影响，本文在实现中进一步加入了激励门限、创新裁剪、单步更新限幅和协方差特征值投影等稳定化处理。

### 4.4 两类惯量辨识方法的特点

从机理上看，ESO + RLS 方案属于显式回归辨识，更新直接、工程可解释性强；增广状态 MEKF 方案则属于状态与参数联合估计，理论一致性更强，但更依赖于估计器建模质量和系统激励。本文同时实现这两类方法，旨在为后续多场景比较研究奠定基础。

## 5. 控制器设计与参数优化

### 5.1 基于四元数误差的 PD 控制器

设姿态误差四元数为 `q_e`，其虚部记为 `q_e,v`，则 PD 控制律写为：

`u = -Kp * q_e,v - Kd * w`

其中 `Kp` 和 `Kd` 分别为比例增益和微分增益。PD 控制器具有结构简单、参数清晰和易于实现的特点，因此被作为本文的基线控制器。

### 5.2 线性自抗扰控制器

本文实现的 LADRC 包括三部分：

1. 跟踪微分器 TD，用于平滑参考姿态；
2. 扩张状态观测器 ESO，用于估计总扰动；
3. 状态误差反馈控制律，用于生成控制力矩。

其核心思想在于将系统中内外不确定因素统一视为总扰动，并通过 ESO 对其进行在线估计与补偿。项目实现中的 LADRC 还支持根据惯量估计结果在线更新内部增益 `b0`。

### 5.3 控制性能评价指标

为实现不同控制器和不同参数配置之间的统一比较，本文选取如下指标作为综合评价依据：

1. 调节时间 `settle_time`；
2. 超调量 `overshoot`；
3. 最终姿态误差 `final_error`；
4. 控制代价 `effort`；
5. 饱和比例 `sat_ratio`。

这些指标分别从动态速度、平稳性、稳态精度、能耗和工程安全性等角度刻画控制性能。

### 5.4 多优化器驱动的 PD 自动调参

为了避免经验调参的主观性，本文构建了多优化器横向比较框架，对 PD 参数 `Kp` 和 `Kd` 进行自动搜索。当前平台支持：

- 网格搜索；
- 随机搜索；
- Nelder-Mead；
- 模拟退火；
- 粒子群优化。

目标函数综合考虑上述多个控制性能指标，并引入基于多随机种子评估的风险敏感项，以降低对单一样本场景的过拟合。

### 5.5 轻量化指标评估路径

在大量参数评估中，若每次都执行完整传感器仿真和滤波链路，将显著增加优化计算开销。针对这一问题，本文为 PD 调参场景设计了轻量化评估路径，仅保留动力学传播和目标函数所需的指标计算，同时保持与完整仿真一致的初始激励与扰动生成机制。这一改进显著降低了优化器横向比较的运行时间。

## 6. 仿真平台设计与实现

### 6.1 平台模块结构

本文实现的仿真平台主要由以下模块组成：

- `dynamics.py`：姿态动力学传播；
- `startracker.py`：星敏感器观测与 Wahba 姿态解算；
- `mekf.py`：姿态与惯量估计；
- `controller.py`、`adrc_controller.py`：控制器实现；
- `estimators.py`：在线惯量辨识；
- `visualization.py`：图像绘制；
- `benchmark.py`：性能基准测试；
- `main.py`：实验组织、数据导出与报告生成。

这种模块划分保证了不同算法模块之间的相对独立性，便于后续替换、扩展或重构。

### 6.2 单次闭环仿真流程

单次闭环仿真的主要流程为：

1. 初始化动力学状态、控制器、滤波器与辨识器；
2. 由真实姿态生成星敏感器姿态观测；
3. 生成陀螺仪测量；
4. 执行 MEKF 预测与更新；
5. 构造角加速度估计；
6. 若启用在线辨识，则更新惯量参数；
7. 控制器计算控制力矩；
8. 对控制力矩施加饱和约束并推进动力学；
9. 记录姿态误差、控制输入、扰动估计和惯量估计结果。

### 6.3 数据输出与结果管理

平台支持自动输出以下结果：

- 单次仿真综合图；
- PD 与 LADRC 对比图；
- 优化器报告图；
- 惯量辨识图；
- 摘要指标 CSV；
- 时序数据 CSV；
- Markdown 汇总报告。

这种统一输出机制显著提高了实验复现和论文写作效率。

## 7. 实验设计与结果分析

本文基于当前平台自动生成的 `output/` 目录结果进行实验分析。代表性实验采用 `full` 运行档位，仿真总时长为 `15.0 s`，时间步长为 `0.03 s`，控制力矩限幅为 `0.5 N·m`。PD 控制器参数通过多优化器调参获得，LADRC 采用当前平台默认带宽型参数设置。在线惯量辨识分别采用 RLS 方案与增广状态 MEKF 方案进行比较。

### 7.5 结果讨论

综合当前实验结果可以得到以下认识：

1. 平台已经能够稳定支撑控制器比较、参数优化和在线辨识三类研究任务；
2. PD 与 LADRC 并不存在绝对优劣，而是在动态速度与稳态鲁棒性方面表现出不同侧重；
3. 多优化器自动调参机制已经能够为 PD 控制器提供稳定可用的参数建议；
4. 在线惯量辨识模块已经具备初步研究价值，但更系统的多场景对比仍是后续工作重点。

## 8. 讨论

从研究平台的角度看，本文当前工作的意义不仅在于单个算法模块的实现，更在于建立了一条完整、可反复调用的实验链路。这使得后续研究可以在相同平台上逐步开展不同控制方法、不同估计算法和不同辨识策略的横向比较。

从算法层面看，当前实验已初步体现出姿态控制问题中的典型权衡关系：快速响应通常意味着更激进的控制输入，而较小稳态误差和更强抗扰能力通常依赖更复杂的观测与补偿机制。PD 与 LADRC 的结果差异说明了这一点。

从参数辨识层面看，当前研究仍停留在对角惯量估计阶段。尽管这对于平台验证与中期研究已经足够，但若要进一步提升理论完整性，后续仍需考虑更复杂参数结构、不同激励方式以及辨识稳定性的系统研究。

## 9. 结论

本文围绕卫星姿态控制中的建模、估计、控制和参数不确定性问题，设计并实现了一套集动力学仿真、传感器建模、MEKF 姿态估计、PD/LADRC 控制、自动调参和在线惯量辨识于一体的综合研究平台。当前平台已经基本打通“动力学—测量—估计—控制—辨识—输出”的完整闭环链路，并能够支持控制器性能比较、参数优化和惯量在线辨识等多类实验研究。

实验结果表明：在当前代表性场景下，PD 控制器具有更快的动态收敛速度，LADRC 在稳态误差、控制能耗和饱和比例方面更具优势；PSO 和 Nelder-Mead 在当前 PD 参数优化问题上表现较优；RLS 与增广状态 MEKF 均能够实现对角惯量在线修正，其中增广状态 MEKF 在当前场景下表现出更小的最终误差范数。上述结果验证了本文所构建平台的有效性，也为后续更深入的多场景统计实验、鲁棒性研究和正式论文写作奠定了基础。
